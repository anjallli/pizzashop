# Pizza Shop

Pizza Shop is a Java-based application developed to manage pizza ordering operations through an interactive and customizable ordering system.
The application allows customers to order pizzas from a menu, customize pizza size and base, add or remove toppings, and dynamically update the total price based on the selected options.

The system also stores customer and order information as text files after the ordering process for record management.
Menu items, pizza prices, and available toppings can be easily updated through external text files, making the application flexible and easy to maintain.

This project demonstrates practical implementation of Java programming concepts including object-oriented programming, file handling, user interaction, and dynamic data management.

Tools & Technologies Used: Java, File Handling, VS Code.
